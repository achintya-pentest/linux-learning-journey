# Linux File Management Lab

## Objective
Practice basic file and directory operations in Linux.

## Tasks Performed

### 1. Created Directory Structure
- Created folders: linux-lab, data, logs, scripts

Command used:
mkdir linux-lab
cd linux-lab
mkdir data logs scripts

---

### 2. Created Files
- Created multiple files inside data folder

Command used:
touch file1.txt file2.txt file3.txt

---

### 3. Copied and Moved Files
- Copied file to logs folder
- Moved file to scripts folder

Commands:
cp file1.txt ../logs/
mv file2.txt ../scripts/

---

### 4. Deleted Files
- Removed files and directories

Commands:
rm file3.txt
rm -r scripts

---

## Challenges Faced
- Understanding relative paths (../)
- Difference between cp and mv

---

## Learning
- Learned how to create, move, copy and delete files
- Understood how directory structure works
- Got comfortable using terminal commands
