# Linux Introduction

## What is Linux?
Linux is an open-source operating system used in servers, cybersecurity, and development.

## Why Linux is important for cybersecurity?
- Most servers run on Linux
- Security tools like Kali Linux are based on Linux
- It gives full control through terminal

## What I learned
Linux is powerful because it allows users to interact directly with the system using commands.
